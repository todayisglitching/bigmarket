## Быстрый старт

### 1. Создайте миграции и примените их:
```bash
python manage.py makemigrations marketplace
python manage.py migrate
```

### 2. Создайте продавца через админку:
- Откройте `/admin/`
- Зайдите в раздел "Продавцы"
- Создайте нового продавца
- Установите пароль (он автоматически захешируется)

### 3. Протестируйте регистрацию клиента:
- Откройте `/auth/client/register/`
- Заполните форму
- После регистрации вы будете перенаправлены на страницу входа

### 4. Протестируйте вход:
- **Клиенты:** `/auth/client/login/`
- **Продавцы:** `/auth/seller/login/`

## URL маршруты

### Веб-формы:
- `/auth/client/register/` - Регистрация клиента
- `/auth/client/login/` - Вход для клиента  
- `/auth/seller/login/` - Вход для продавца
- `/auth/logout/` - Выход

### API (без изменений):
- `/api/auth/client/register/` - Регистрация через API
- `/api/products/` - Управление продуктами

## Безопасность

✅ Все пароли автоматически хешируются через Django  
✅ Пароли никогда не хранятся в открытом виде  
✅ Используется безопасный алгоритм PBKDF2  
✅ Токены генерируются автоматически  

## Важно

- **Продавцы регистрируются только через админку/БД**
- **Клиенты могут регистрироваться через веб-форму или API**
- **Поле `checked` у продуктов изменяется только администратором**
